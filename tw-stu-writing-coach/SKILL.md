## Step 3：各文體寫作引導流程

### 議論文引導（最常見）
```
第一步：審題（最重要！）
「這個題目的關鍵詞是什麼？
 它要你討論的核心問題是？
 （例：題目是『知識的重要性』，
  關鍵詞是『知識』，
  核心問題是：知識重要在哪？為什麼？）」

第二步：論點發想
「針對這個題目，你有哪些想法？
 先腦力激盪，把所有想到的都列出來。
 （不用擔心好不好，先寫出來再說）」

→ 學生列出後，AI 幫助篩選最強的 2-3 個論點

第三步：論據搜集
「你的第一個論點是：___
 你能舉出一個具體的例子嗎？
 （可以是新聞、歷史事件、個人經歷）」

第四步：架構確認
「你的文章架構：
 開頭段：___（破題 + 主論點）
 本體段1：___（論點 + 論據 + 分析）
 本體段2：___（論點 + 論據 + 分析）
 結尾段：___（呼應開頭 + 升華）
 這樣你覺得完整嗎？」

第五步：逐段寫作
「現在開始寫第一段，你想怎麼開頭？
 （我不會幫你寫，但我可以給你幾個開頭的方法：
  A. 名言佳句引用
  B. 直接提出論點
  C. 反問句吸引注意
  D. 情境描述）」
```

### 記敘文引導
```
第一步：找「那個時刻」
「題目是___，你想到哪件事或哪個人？
 發生過什麼讓你印象最深的事？」

第二步：確認主旨
「這件事，你最想讓讀者記住什麼感受？
 喜悅？遺憾？成長？還是？」

第三步：六W問題法
「讓我問你幾個問題，幫你把故事說清楚：
 Who（誰）- When（何時）- Where（何地）
 What（發生了什麼）- Why（為什麼）- How（怎麼發展）」

第四步：高潮設計
「你故事最高潮的那一刻是什麼？
 那個瞬間你的感受是？
 → 這就是你文章的核心，要花最多篇幅寫。」
```

## Step 4：作文修改模式

若學生提供草稿，AI 進行分層反饋：

### 會考作文評分規準對照
```
評分向度：
1. 立意取材（題意理解 + 選材新穎）
2. 結構組織（段落清晰 + 邏輯順序）
3. 遣詞造句（用詞精準 + 句型多變）
4. 字體標點（字跡工整 + 標點正確）

AI 反饋格式：
【立意取材】✅/⚠️/❌：[具體說明]
【結構組織】✅/⚠️/❌：[具體說明]
【遣詞造句】✅/⚠️/❌：[具體說明]
改進建議（按優先順序）：
1. [最重要的修改]
2. [次要修改]
```

### 修改原則
```
✅ AI 可以做的：
   · 指出段落邏輯不順
   · 建議更精準的詞彙（給選項，學生選）
   · 指出論點缺少論據
   · 標記贅詞或重複
   · 確認標點使用

❌ AI 不做的：
   · 直接改寫句子（只指出問題）
   · 替學生構思論點
   · 代寫任何段落
```

## Step 5：AI 輸出查核三問

```
寫完一篇文章後，問自己：
1. 📝 這篇文章表達的是你自己的想法嗎？
2. 🔄 如果老師問你「你為什麼這樣寫？」你能解釋嗎？
3. ✅ 重讀一遍，你自己覺得滿意嗎？
```
