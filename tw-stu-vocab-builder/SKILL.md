## Step 3：文言文詞彙模組

### 詞彙解析格式
```
【詞彙】之（虛詞）

📖 基本用法（5 種）：
1. 代詞：「之」代替前面提到的人或事
   例：「師道之不傳也久矣」→「之」= 的（結構助詞）
2. 結構助詞「的」
3. 動詞（去/到）：「吾欲之南海」→「之」= 去
4. 音節助詞（無義）
5. 賓語前置標誌

🔍 例句比較：
·「學而時習之」（代詞）
·「大道之行也」（助詞）
·「送孟浩然之廣陵」（動詞）

✏️ 練習：
「以下哪個「之」的用法和「師道之不傳也久矣」相同？
A. 「宋之丁氏」 B. 「往之女家」 C. 「學而時習之」
→ 試著分析每個句子再回答。」
```

### 成語互動學習
```
成語：「半途而廢」

📜 出處：《禮記・中庸》
💡 原意：做事到了中途就停下來，不繼續完成
🌟 現代語境：「他學吉他學了兩個月就半途而廢。」

⚠️ 常見錯誤用法：
❌「他終於完成了，真是半途而廢！」（意思相反！）

✏️ 你來造個句子，我來評估用法是否正確。」
```

## Step 4：閱讀理解策略模組

### 素養導向閱讀題解題框架
```
台灣素養題特點：
・沒有固定答案範圍（不考背誦）
・考「理解」和「應用」
・常見題型：
  1. 作者意圖題（這篇文章主要在說什麼？）
  2. 推論題（根據文章，可以得知…）
  3. 評估題（對文章觀點提出看法）
  4. 跨文本比較（兩篇文章的共同點/不同點）

解題步驟：
Step 1：先讀題目，再讀文章
  → 帶著問題讀，不是讀完再猜
Step 2：找關鍵句（通常在段落首尾）
Step 3：排除選項法（找出一定錯的選項）
Step 4：答案要在文章找根據，不能憑感覺
```

### 引導式閱讀練習
```
[貼入一段文章後]

AI 不直接出題，先問：
「讀完這段文字，你的第一個感受是什麼？
 有沒有讓你印象最深的一句話？」

→ 學生回答後，AI 追問：
「你說這句話讓你印象深刻，
 作者在這裡用了什麼技巧？
 這句話和上下文有什麼關係？」
```

## Step 5：詩詞欣賞模組

```
篇目：〈靜夜思〉李白

🎵 誦讀指引：
「先不管意思，把詩讀出聲音。
 你有沒有感受到某種情緒？說說看。」

🔍 字詞解析：
「疑」→不確定，以為
「霜」→霜（但白居易用「霜」比喻月光）

💡 意象分析：
「月光、霜、舉頭、低頭」——
AI 問：「這些動作有什麼順序？
        詩人抬頭和低頭，心情一樣嗎？」

🌏 脈絡連結：
「你有沒有在外地（或很久沒回家）
 看著月亮想家的感覺？
 這跟李白的感受有什麼相似的地方？」
```
