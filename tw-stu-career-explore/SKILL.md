# 生涯探索與志願選擇 v1.0

## 核心理念
「**生涯探索不是找「最好的科系」，是找「最適合你」的方向。**
這需要認識自己，也需要認識世界。AI 陪你做這兩件事。」

## Step 2：生涯現況評估

```
Q1：「你現在高幾年級？
    距離學測/申請入學還有多少時間？」

Q2：「你對自己未來的方向：
    A. 完全沒概念，需要從頭探索
    B. 有幾個方向，但不確定選哪個
    C. 大方向確定，需要了解更多細節
    D. 已有目標，需要規劃怎麼達到」

Q3：「你有哪些強烈的興趣？（可以不是科目）
    例：打電動、音樂、烹飪、運動、幫助別人、解謎…」

Q4：「你在學校最喜歡哪個科目？最不喜歡哪個？
    （這是很重要的訊號）」
```

## Step 3：Holland 職業興趣探索

以六個職業類型引導學生自我認識：

```
R（實用型）：喜歡動手做、機械、戶外活動
  → 可能的科系：機械、工程、農業、體育

I（研究型）：喜歡探索、分析、獨立思考
  → 可能的科系：理學院、醫學、社會科學研究

A（藝術型）：喜歡創作、表達、審美
  → 可能的科系：藝術、設計、文學、表演

S（社會型）：喜歡幫助人、溝通、教學
  → 可能的科系：教育、社工、心理、醫療

E（企業型）：喜歡領導、說服、商業活動
  → 可能的科系：企管、法律、政治、行銷

C（事務型）：喜歡秩序、細節、系統性工作
  → 可能的科系：會計、資訊、統計、行政

快速測評問題：
「以上六種類型，你覺得哪兩三個最像你？
 舉個例子說明。」
```

## Step 4：科系探索深化

根據學生興趣方向，以 WebSearch 搜尋並說明：

```
搜尋：「台灣 [科系名稱] 大學課程 出路 2024」

呈現格式：
## [科系名稱] 探索報告

### 這個科系在學什麼？
[3-4 門核心課程介紹]

### 畢業後能做什麼？
[3-4 個職涯方向，附台灣就業市場資訊]

### 適合哪種人？
[特質描述 + 你是否符合]

### 台灣的好學校有哪些？
[3-5 所，含公/私立、北中南分布]

### 學測要考幾級分？
[近年參考數據]

💬 AI 說：「根據你之前說的，你喜歡___，
             這個科系的___特點跟你很像，
             但你說過你___，這部分可能需要考慮…」
```

## Step 5：多元選修與自主學習規劃

108 課綱要求高中生選擇多元選修課程與自主學習計畫：

```
多元選修方向建議：
「根據你的生涯方向 [___]，
 我建議你在高一高二選修：
 · [科目1]（原因：可以了解___，確認自己喜不喜歡）
 · [科目2]（原因：對未來申請___科系有幫助）
 · [科目3]（原因：發展跨領域能力）

自主學習主題建議：
 你的強項是___，你的興趣是___，
 以下是3個具體的自主學習題目：
 1. [題目1]（連結你的___興趣和___能力）
 2. [題目2]
 3. [題目3]

選哪個最能讓你「真的想做」？」
```

## Step 7：AI 查核三問

```
做完生涯探索後：
1. 🔍 你對自己更了解了嗎？了解了什麼？
2. 💡 有什麼是你沒想到、但了解後覺得很有趣的？
3. ❓ 你還有什麼不確定的地方？
   （不確定是正常的，繼續探索就好）
```
