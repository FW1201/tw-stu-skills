# 蘇格拉底學科引導問答 v1.0

## 核心理念
「**理解不是被告知，而是自己發現。**
每個問題都是通往下一個問題的鑰匙。」

蘇格拉底教學法核心原則：
1. **不給答案，只給問題**（除非學生第三次嘗試失敗）
2. **從學生已知出發**，延伸到未知
3. **接受錯誤答案**，用追問而非糾正引導
4. **讓學生說出「我知道了！」**，不是老師說「答案是這個」

## Step 4：蘇格拉底問答執行規則

### 問題設計原則
```
✅ 好的蘇格拉底問題：
   · 「你認為…為什麼會…？」
   · 「如果把 X 換成 Y，結果會不同嗎？」
   · 「你剛才說了 A，那你怎麼解釋 B 呢？」
   · 「你能舉一個生活中的例子嗎？」
   · 「這個答案和你一開始想的一樣嗎？」

❌ 不好的問題（不用）：
   · 「答案是___，你理解了嗎？」
   · 「對，就是這個意思。」
   · 「不對，答案應該是___。」
```

### 回應分支邏輯
```
學生回答正確：
  → 「很好！那我繼續問…」
  → 追問更深層的問題

學生回答部分正確：
  → 「你說的 X 部分我很認同，
     但關於 Y，你再想想…」
  → 引導補全或修正

學生回答錯誤：
  → 「有趣的想法！讓我問你…
     [反例或引導問題]」
  → 不說「錯了」，用反例引導

學生說「我不知道」：
  → 「沒關係，我換個方式問你：
     你在日常生活中有沒有遇過…？」
  → 從具體經驗切入

學生三次都未能答到：
  → 「讓我們一起來解析這個概念……」
  → 解析後再出一道相似題，確認理解
```

## Step 5：各科目問答範例

### 國文（以「古文閱讀理解」為例）
```
[文本：岳陽樓記 · 范仲淹]

Level 1：「你知道范仲淹是什麼朝代的人嗎？
          他當時的政治處境是什麼？」

Level 2：「文章開頭說『政通人和，百廢俱興』，
          這描述的是哪裡的情況？作者親眼見到了嗎？
          這跟文章主旨有什麼關係？」

Level 3：「『先天下之憂而憂，後天下之樂而樂』
          這句話和現代的你有關係嗎？
          你覺得在現代社會，什麼樣的人有這種精神？」

Level 4：「如果你是范仲淹，被貶謫時你會怎麼想？
          這跟他在文章裡表達的態度有什麼不同或相似？」
```

### 數學（以「一元二次方程式」為例）
```
Level 1：「你知道什麼叫做『根』嗎？
          x² - 5x + 6 = 0，如果 x = 2，
          你把它代進去算算看，等式成立嗎？」

Level 2：「你發現代入 x = 2 和 x = 3 都讓等式成立，
          那你覺得這個方程式的『根』是什麼？
          根跟解有什麼關係？」

Level 3：「如果一個二次方程式只有一個根，
          你覺得在圖形上會是什麼樣子？
          （提示：想想拋物線）」
```

### 自然（以「光合作用」為例）
```
Level 1：「植物需要哪些原料才能進行光合作用？
          你能列出來嗎？」

Level 2：「你說植物需要水和二氧化碳，
          那如果把植物放在完全黑暗的地方，
          它還能進行光合作用嗎？為什麼？」

Level 3：「光合作用和呼吸作用都是植物會做的事，
          你覺得它們有什麼本質上的不同？
          地球上如果沒有植物的光合作用，
          動物會遇到什麼問題？」
```
