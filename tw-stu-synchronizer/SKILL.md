# K-12 學習套組客製化工具 v1.0

## 核心目的
「**讓每個 tw-stu-* Skill 都知道「你是誰」。**
設定一次，全套組都變成你的專屬工具。」

## Step 2：系統性問卷（15 題）

### 基本資料
```
Q1：「你的名字或暱稱？（讓 AI 稱呼你）」
Q2：「你現在是幾年級？
    國一/國二/國三/高一/高二/高三/其他」
Q3：「你就讀的是：
    A. 公立國中/高中   B. 私立國中/高中
    C. 科技高中職       D. 其他」
```

### 學習目標
```
Q4：「你最近的重要考試目標是？
    A. 國中教育會考   B. 學科能力測驗（學測）
    C. 分科測驗        D. 段考/平時考
    E. 目前沒有特定考試目標」
Q5：「你的目標成績/等級是？（請說明）」
Q6：「你最想進步的科目是？（前三名）」
Q7：「你最弱、最需要加強的科目是？（誠實）」
```

### 學習風格
```
Q8：「你記憶最好的方式？
    A. 看圖/圖解（視覺型）
    B. 聽解說（聽覺型）
    C. 實際做題（動覺型）
    D. 寫筆記整理（書寫型）」
Q9：「你的注意力持續時間？
    A. 20分鐘以內就需要換      B. 40-50分鐘
    C. 1小時以上」
Q10：「你偏好的學習語言？
     A. 全中文                  B. 中英混合（可接受技術詞彙英文）」
```

### 個性與習慣
```
Q11：「遇到不會的題目，你通常：
     A. 馬上查答案/問人
     B. 自己想一會兒再查
     C. 先跳過，之後回來」
Q12：「你希望 AI 對你的語氣是？
     A. 像朋友（輕鬆口語）
     B. 像老師（正式引導）
     C. 像教練（直接指出問題）」
Q13：「你對使用 AI 輔助學習的態度？
     A. 工具派（效率優先）
     B. 學習派（理解優先）
     C. 還在摸索」
```

### 特殊需求
```
Q14：「你有沒有特別的學習困難？
     A. 沒有
     B. 閱讀理解困難
     C. 數學計算困難
     D. 文字記憶困難
     E. 考試焦慮
     F. 其他：___」
Q15：「有什麼是你希望 AI 特別注意的？
     （自由填寫）」
```

## Step 3：生成 student-profile.md

```markdown
# 學習者設定檔
# 建立日期：[YYYY-MM-DD]
# tw-stu-synchronizer v1.0

---

## 基本資料
- 稱呼：[名字]
- 年級：[年級]
- 學校類型：[類型]

## 學習目標
- 近期目標考試：[考試名稱]
- 目標成績：[目標]
- 優先科目：[科目列表]
- 弱科：[弱科列表]

## 學習風格偏好
- 記憶方式：[視覺/聽覺/動覺/書寫]
- 注意力持續：[N]分鐘
- 語言偏好：[中文/中英混合]

## 互動偏好
- 遇到困難時：[立即查/自己想/跳過]
- 期望語氣：[朋友/老師/教練]
- AI 使用態度：[工具派/學習派]

## 特殊設定
- 學習困難：[若有]
- 特別注意事項：[若有]

## 各 Skill 客製化參數
### tw-stu-exam-prep
- 目標考試：[考試]
- 優先弱科：[弱科1]、[弱科2]

### tw-stu-socratic
- 問題深度：[基礎/中等/進階]
- 科目重點：[科目列表]

### tw-stu-concept-viz
- 偏好圖形類型：[類型]

### tw-stu-writing-coach
- 常見文體：[文體]
- 寫作弱點：[弱點]
```

## Step 4：已啟動的自訂項目說明

輸出清單：
```
✅ 已客製化的項目（[N] 項）：

1. AI 稱呼你為「[名字]」
2. 所有說明自動調整為[年級]程度語言
3. 練習題優先出現[弱科]相關題目
4. 語氣調整為[偏好語氣]風格
5. 注意力模組：每[N]分鐘提醒換科目
6. [特殊設定]已啟用

📁 設定檔已存入：./student-profile.md
   （所有 tw-stu-* Skills 啟動時會自動讀取此檔案）
```

---

## Step 5：更新設定

設定檔建立後，可隨時更新：
```
「更新我的設定」→ 進入特定問題修改模式
「完全重設」→ 重新完成全部問卷
「查看我的設定」→ 顯示目前設定摘要
```
