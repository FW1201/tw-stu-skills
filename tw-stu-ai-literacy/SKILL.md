# AI 工具正確使用法 v1.0

## 核心理念
「**AI 是你的思考夥伴，不是你的代筆工具。**
用 AI 讓自己更聰明，而不是讓 AI 替你思考。」

## Step 3：AI 基礎觀念模組

### 模組 A：AI 的能力與限制
```
AI 做得好的事：
✅ 整理、歸納已知資訊
✅ 解釋概念（但要自己確認）
✅ 提供寫作結構框架
✅ 產生練習題讓你作答
✅ 翻譯、語法修正

AI 做不好的事：
❌ 最新資訊（訓練資料有截止日）
❌ 數字計算（常錯，必須驗算）
❌ 台灣本土特定資料（考試規範等）
❌ 你的個人經歷和感受
❌ 保證「這個答案一定正確」

⚠️ AI 的致命弱點：幻覺（Hallucination）
AI 有時會「自信地說錯誤的事」。
→ 任何重要資訊都必須用其他來源查證。
```

### 模組 B：三種使用模式比較
```
❌ 誤用模式：「幫我寫一篇關於環保的作文」
   → AI 生成，學生直接交
   → 後果：沒有學習、被老師發現、學術誠信問題

⭕ 輔助模式：「我想寫環保作文，可以幫我列出3個論點嗎？
              我自己會選一個來寫。」
   → AI 給方向，學生自己選擇並發展
   → 效果：有學習，成果是自己的

✅ 最佳模式：「我在寫關於環保的段落，這是我的草稿：___
              你覺得論點夠清楚嗎？哪裡可以加強？」
   → AI 給反饋，學生自己修改
   → 效果：深度學習，AI 是真正的夥伴
```

## Step 4：提示詞設計工作坊

### 提示詞的四個要素
```
1. 角色（Role）：「你是一位○○老師」
2. 任務（Task）：「請幫我___」
3. 脈絡（Context）：「我是高一學生，正在學___」
4. 限制（Constraint）：「請用國中生能懂的語言」
```

### 練習：從壞提示詞到好提示詞
```
練習 1
❌ 壞的：「幫我解釋光合作用」
✅ 好的：「我是國二學生，剛學完光合作用，
          但還是不懂為什麼一定要有光。
          請用一個日常生活的例子解釋給我聽。」

練習 2
❌ 壞的：「寫一首關於秋天的詩」
✅ 好的：「我要寫一首關於秋天的詩，主題是『思念』。
          請問七言絕句要怎麼押韻？
          給我3個可以押韻的字組就好，我自己來填詞。」
```

## Step 5：AI 輸出查核五步驟

```
當你拿到 AI 的回答，一定要做這 5 件事：

1. 🔍 查來源：AI 提到的事實有出處嗎？用 WebSearch 查證。
2. 🧮 算數字：AI 提供的數字、統計，自己驗算一遍。
3. 🇹🇼 台灣化：是台灣的制度嗎？AI 常把台灣跟其他地方搞混。
4. ✍️ 自己寫：把 AI 的說明用自己的話重新表達。
5. 💬 問老師：如果你不確定，問老師比問 AI 更可靠。
```

## Step 6：學術誠信界線說明

```
台灣教育部 AI 使用指引（2024）重點：

✅ 允許的 AI 使用：
   · 搜集資料靈感（但需標明 AI 輔助）
   · 語法修正（需標明）
   · 理解概念（學習用途）

❌ 不允許的 AI 使用：
   · 直接提交 AI 生成內容為自己的作品
   · 考試中使用 AI
   · 用 AI 代替自己思考的作業

如何標明 AI 使用：
「本文部分段落經 AI 工具輔助修改，
 核心觀點與寫作為本人完成。」
```

## Step 7：AI 倫理思考練習（蘇格拉底式）

針對爭議情境，AI 不給答案，只問問題：

```
情境：「我用 AI 把英文作業翻譯成中文，算作弊嗎？」

AI 的回應方式：
「這是個好問題！我想先問你幾個問題：
1. 你的老師出這份作業的目的是什麼？
2. 你用翻譯後，你學到了什麼？
3. 如果每個同學都這樣做，對整個班的學習有什麼影響？
4. 如果老師知道你這樣做，你覺得他/她會怎麼想？
說說你自己的看法？」
```
