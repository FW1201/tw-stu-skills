## Step 3：科學學習方法教學

### 認知科學三大原則
```
原則 1：間隔效應（Spaced Repetition）
今天學的東西：
  ✅ 明天複習一次（5分鐘）
  ✅ 3天後複習一次（5分鐘）
  ✅ 7天後複習一次（5分鐘）
  ✅ 14天後複習一次（5分鐘）
  → 每次複習都在「快忘記」之前，效果最好

原則 2：主動回憶（Active Recall）
  ❌ 重讀課本 → 感覺懂了但其實沒真正記住
  ✅ 蓋上課本，用自己的話說出來 → 真正測試記憶

原則 3：交錯練習（Interleaving）
  ❌ 一次讀完全部數學，再讀全部英文
  ✅ 每 40 分鐘換一個科目
  → 大腦需要切換，反而記憶更深
```

## Step 4：個人化計畫生成

根據學生資料，輸出：

```markdown
## [學生名] 的 [N] 週備考計畫

### 📊 科目優先級
🔴 高優先：[科目]（投入 40% 時間）
🟡 中優先：[科目]（投入 35% 時間）
🟢 保溫：[科目]（投入 25% 時間）

### 📅 每日學習排程（平日）
17:00-17:15 到家調整，吃點心
17:15-17:55 [高優先科目]（主動回憶練習）
17:55-18:00 休息 5 分鐘
18:00-18:40 [中優先科目]（做題）
18:40-19:30 晚餐
19:30-20:10 [高優先科目]（複習昨天學的）
20:10-20:50 [保溫科目]（快速瀏覽）
20:50-21:00 整理今日筆記（5分鐘）

### 📅 週末強化排程
早上 2 小時：本週最弱單元集中攻
下午 2 小時：模擬測驗（計時練習）
晚上 1 小時：整理錯題本

### 🔄 間隔複習提醒
周一學的→周二複習→周四再看→下周一最後確認

### 📌 本週第一步（明天就可以開始）
1. [具體行動]
2. [具體行動]
```

## Step 5：MCP Connector 整合

### Google Calendar
```

```
若使用者說「記錄我的進度」：
→ 建立每日學習記錄格式：
  日期 | 科目 | 讀了什麼 | 用什麼方法 | 理解程度(1-5)
```

## Step 6：AI 查核三問

```
執行計畫一週後，檢查：
1. 📋 你有照著計畫執行嗎？（誠實說）
2. 🔍 哪個科目進步最多？哪個還是卡關？
3. 🔄 計畫需要調整什麼？（太緊/太鬆/順序不對）
